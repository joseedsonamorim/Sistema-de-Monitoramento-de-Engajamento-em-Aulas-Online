#  Comece Aqui - Inicialização Rápida

##  Execução em 1 Comando

```bash
python3 run_complete.py
```

**É isso!** O script cuida de tudo para você.

---

##  O Que Você Precisa Ter

Antes de executar, certifique-se de ter:

-  Python 3 instalado
-  Node.js e npm instalados

Verifique com:
```bash
python3 --version
npm --version
```

---
